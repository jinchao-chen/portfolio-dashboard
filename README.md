# portfolio-dashboard

This packages creates a dashboard for trading212 trading activities. 

## how to set up


- download the transaction history from Trading212. It is explained in this [post](https://community.trading212.com/t/new-feature-export-your-investing-history/35612) on how to download it in 'csv' format.
- If you prefer to run the app locally, you will need to first install [streamlit](https://streamlit.io). 
- A web app hosted by streamlit is readily available for use: [web_app](https://share.streamlit.io/jinchao-chen/portfolio-dashboard/streamlit/web_app.py)
